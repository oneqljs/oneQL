运行环境
-----

#### 默认环境

| 环境 | 环境变量 | 说明 |
| ----- | --------- | -- |
| dev | dev | 本地开发环境 |
| fat | fat | fat测试环境 |
| prod | prod | 生产环境 |
| test | test | 单元测试 |


#### 自定义环境

<p>由于环境变量和发布系统可能有关，不同公司内部环境变量也许有区别，所以可以自行修改项目根路径下的 app.config.js 中的 env变量，然后可自行处理对应env的逻辑。</p>

<p>* prod环境 会默认会关闭 graphql调试面板</p>
